# reports

ITコンサルチームの検討資料・レポート類を共有するための静的サイトです。

公開URL: https://isense-cs.github.io/reports/

## 運用方針

- URLを知っている方のみが閲覧する前提(GitHub Pages公開・robots noindex指定)
- 機密度の高い情報は掲載しない
- 新しいレポートを追加する場合は、HTMLをルートに置き `index.html` から該当ファイルへリンクを追加

## ローカル確認

```powershell
# 直接ブラウザで index.html を開く
Start-Process .\index.html
```
